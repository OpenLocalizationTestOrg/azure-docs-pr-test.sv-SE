# <a name="turn-on-high-contrast-or-change-the-theme-in-the-azure-portal"></a>Aktivera hög kontrast eller ändra temat i Azure-portalen
Om du behöver mer kontrast eller om du vill ändra färgschema i Azure portal, gå till portalinställningar man gör ändringen. 

## <a name="turn-on-high-contrast"></a>Aktivera hög kontrast
1. Högst upp i den [Azure-portalen](https://portal.azure.com)väljer **inställningar**. 

    ![Skärmbild som visar portalinställningar redskap ikon i Azure-portalen](./media/azure-portal-change-theme-high-contrast/azure-portal-settings-icon.png)
1. Välj **vitt** eller **svart**.

    ![Skärmbild som visar alternativ för hög kontrast i inställningarna för Azure-portalen](./media/azure-portal-change-theme-high-contrast/azure-portal-highcontrast-options.png)
1. Välj **Använd**.

## <a name="change-theme"></a>Ändra tema
1. Högst upp i den [Azure-portalen](https://portal.azure.com)väljer **inställningar**.

    ![Skärmbild som visar portalinställningar redskap ikon i Azure-portalen](./media/azure-portal-change-theme-high-contrast/azure-portal-settings-icon.png)
1. Välj ett tema.

    ![Skärmbild som visar alternativ för tema i inställningarna för Azure-portalen](./media/azure-portal-change-theme-high-contrast/azure-portal-theme-options.png)
1. Välj **Använd**.
