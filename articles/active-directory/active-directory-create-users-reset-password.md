# <a name="reset-a-password-for-a-user-in-the-azure-classic-portal"></a>Återställa ett lösenord för en användare i den klassiska Azure-portalen
> [!div class="op_single_selector"]
> * [Azure Portal](active-directory-users-reset-password-azure-portal.md)
> * [Klassisk Azure-portal](active-directory-create-users-reset-password.md)
>
>

Azure Active Directory (AD Azure) gör enklare att återställa en användares lösenord om de har glömt och har låsts ut eller när du utför rutinmässiga uppdateringar. 

> [!IMPORTANT]
> Microsoft rekommenderar att du hanterar Azure AD via [Azure AD administratörscenter](https://aad.portal.azure.com) på Azure Portal istället för via den klassiska Azure-portalen som nämns i den här artikeln. Att återställa en användarlösenord i administrationscentret för Azure AD, se [återställa lösenordet för en användare i Azure Active Directory](active-directory-users-reset-password-azure-portal.md).

1. Öppna din katalog.
2. Välj fliken **Användare** och välj sedan visningsnamnet för den användare som du vill ändra.
3. Välj i kommandofältet **Återställ lösenord**.
4. I dialogrutan Återställ lösenord klickar du på **återställa**.
5. Välj bockmarkeringen för att återställa lösenordet är klar.
