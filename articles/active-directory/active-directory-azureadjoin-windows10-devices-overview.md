# <a name="windows-10-for-the-enterprise-ways-to-use-devices-for-work"></a>Windows 10 för företaget: Sätt att använda enheter för arbete
Windows 10 ger dig möjlighet att utnyttja Azure Active Directory (AD Azure). Du kan ansluta Windows 10-enheter till Azure AD så att användarna kan logga in till Windows med hjälp av Azure AD-konton eller genom att lägga till sina Azure-ID: N att få åtkomst till företagsappar och resurser.
