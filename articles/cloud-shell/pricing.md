# <a name="pricing"></a>Prissättning

## <a name="compute-cost"></a>Beräkna kostnaden
Azure Cloud Shell körs på en dator som tillhandahålls gratis, men kräver en bifogad fil i Azure resursen toouse. 

## <a name="storage-cost"></a>Kostnad för datalagring
Molnet Shell skapar en standardbild 5GB i din filresursen toopersist $HOME-katalogen. Azure-filresurser innebära regelbundna kostnader.

Kontrollera [här för mer information om Azure-filer kostnader](https://azure.microsoft.com/pricing/details/storage/files/).
