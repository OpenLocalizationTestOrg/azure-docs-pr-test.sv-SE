# <a name="azure-app-service-security"></a>Azure Apptjänst-säkerhet
Azure Apptjänst uppfyller viktiga branschstandarder för säkerhet och tillförlitlighet för att ge en säker plattform för våra kunder. Vi har flera lätt att använda funktioner för att skydda både inkommande och utgående trafik för din app. Vi kan också hjälpa kunderna säker deras programkod genom att tillhandahålla externt tillhandahålla funktionalitet för genomsökning av ditt webbprogram för säkerhetsproblem.

[!INCLUDE [app-service-blueprint-security](../../includes/app-service-blueprint-security.md)]

Azure Apptjänst använder samma program mot skadlig kod som används av Azure-molntjänster och virtuella datorer. Läs mer om detta finns i vår [program mot skadlig kod dokumentationen](../security/azure-security-antimalware.md).
