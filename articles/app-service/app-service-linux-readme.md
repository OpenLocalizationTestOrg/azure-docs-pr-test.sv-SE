# <a name="web-app-on-linux-documentation"></a>Webbprogrammet på Linux-dokumentation
Web App på Linux är för närvarande i förhandsversion och ger kunder möjlighet att köras sina webbprogram på en Linux-plattformen. Detta ger bättre programkompatibilitet för vissa typer av program och gör det enklare att migrera befintliga webbprogram finns på en Linux-plattformen någon annanstans till Azure App Service.

Kunder kan skapa webbappar i Linux endast i en Linux-aktiverad App Service-Plan och denna App Service-Plan kan bara vara värd för Linux-baserade webbappar. 

> Priset beräknas genom halving antal beräkningstimmar som används av din Programtjänstplan Public Preview priser för webbprogrammet på Linux är 50% av priset.
> 
> Inga SLA erbjuds i offentliga förhandsversionen.
