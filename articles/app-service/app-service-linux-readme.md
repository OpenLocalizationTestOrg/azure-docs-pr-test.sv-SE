# <a name="web-app-on-linux-documentation"></a>Webbprogrammet på Linux-dokumentation
Web App på Linux är för närvarande i förhandsversion och kan kunder toorun sina webbprogram internt på en Linux-plattformen. Detta ger bättre programkompatibilitet för vissa typer av program och gör det enklare toomigrate befintliga webbprogram finns på en Linux-plattformen någon annanstans till Azure App Service.

Kunder kan skapa webbappar i Linux endast i en Linux-aktiverad App Service-Plan och denna App Service-Plan kan bara vara värd för Linux-baserade webbappar.
