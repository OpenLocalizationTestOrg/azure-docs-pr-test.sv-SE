# <a name="get-started-with-hello-box-connector"></a>Kom igång med hello rutan connector
Anslut tooBox och skapa filer och ta bort filer. I rutan kan du:

* Skapa ditt företag flödet baserat på hello data du får från rutan. 
* Använd utlösare när en fil skapas eller uppdateras.
* Använd åtgärder som att kopiera en fil, ta bort en fil med mera. De här åtgärderna få svar och hello utdata gör tillgängligt för andra åtgärder. När en fil ändras i rutan, kan du dra filen och e-den med hjälp av Office 365.

Du kan komma igång genom att skapa en logikapp nu, se [skapa en logikapp](../logic-apps/logic-apps-create-a-logic-app.md).
