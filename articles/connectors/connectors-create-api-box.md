# <a name="get-started-with-the-box-connector"></a>Kom igång med rutan connector
Ansluter till Box och skapa filer och ta bort filer. I rutan kan du:

* Skapa ditt företag flödet som baseras på de data som du får från rutan. 
* Använd utlösare när en fil skapas eller uppdateras.
* Använd åtgärder som att kopiera en fil, ta bort en fil med mera. De här åtgärderna få svar och utdata gör tillgängligt för andra åtgärder. När en fil ändras i rutan, kan du dra filen och e-den med hjälp av Office 365.

Du kan komma igång genom att skapa en logikapp nu, se [skapa en logikapp](../logic-apps/logic-apps-create-a-logic-app.md).

## <a name="connector-specific-details"></a>Connector-specifik information

Visa alla utlösare och åtgärder som definierats i swagger och även se några gränser i den [connector information](/connectors/box/).
