[Hantera hemligheter för Azure-appar](https://channel9.msdn.com/events/Build/2016/P456)

Varje allvarliga app som du distribuerar i Azure har kritiska hemligheter – anslutningssträngar, certifikat, nycklar. Löjliga fel vid hantering av dessa hemligheter leder till allvarliga följder – minnesläckor, avbrott, efterlevnad överträdelser. Samma som flera senaste undersökningar, orsaka Löjliga misstag fyra gånger större dataintrång än motståndare. I den här sessionen gå vi igenom några rekommendationer för att hantera dina viktiga apphemligheter. Följande rekommendationer kan verka gemensamma mening, men många utvecklare underlåter dem. Vi också gå igenom hur du använder Azure Key Vault för att implementera de bästa praxis. En fördel följande dessa metoder kan visa du följer standarder, till exempel SOC. 10 minuter sessionen är nivå 100 och de gäller för alla molnappar som du utvecklar på valfri plattform. Resten är nivå 200 300 och fokuserar på appar som du skapar på Azure-plattformen.
