# <a name="personnel-security-ps"></a>PERSONALSÄKERHET (PS)

> [!NOTE]
> De här kontrollerna definieras av NIST och USA Handelsdepartementet som en del av NIST Special Publication 800-53 version 4. Läs NIST 800 53 Rev. 4 för information om hur du testar procedurer och riktlinjer för varje kontroll.

## <a name="nist-800-53-control-ps-1"></a>NIST 800 53 kontrollen PS-1

#### <a name="personnel-security-policy-and-procedures"></a>Säkerhetsprincip för personal och procedurer

**PS-1** organisation utvecklar, dokument och sprider till [tilldelning: personal som definierats av organisationen eller roller] en säkerhetsprincip för personal som åtgärdar syfte, scope, roller, ansvar, management åtagande, samordning bland organisationens entiteter och efterlevnad; och procedurer för att underlätta genomförandet av personal säkerhetsprincip och associerade personal säkerhetsåtgärder; och granskar och uppdaterar den aktuella personal säkerhetsprincipen [tilldelning: organisation definierats frekvens]; och procedurer för PERSONALSÄKERHET [tilldelning: organisation definierats frekvens].

**Ansvarsområden:**`Customer Only`

|||
|---|---|
| **Kunden** | Kundens på företagsnivå personal säkerhetsprinciper och procedurer kan vara tillräcklig för att åtgärda den här kontrollen. |
| **Providern (Microsoft Azure)** | Inte tillämpligt |
