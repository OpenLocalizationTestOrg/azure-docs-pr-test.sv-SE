# <a name="how-to-log-a-security-event-support-ticket"></a>Logga ett supportärende för säkerhet händelse

1. Gå till [utgivarens Support](https://support.microsoft.com/en-us/getsupport?wf=0&tenant=ClassicCommercial&oaspworkflow=start_1.0.0.0&locale=en-us&supportregion=en-us&pesid=16230&ccsid=636450758943226673) och logga in med autentiseringsuppgifterna för ditt Microsoft.
2. Markera ”säkerhetshändelse” som Problem och välja mellan ”säkerhetsincident” och ”säkerhetsproblem” kategorier.

    ![Händelsetyp och Definition][1]

3. När du väljer problemtyp och kategori, klickar du på den '**startbegäran**-knappen. Ange följande information som hjälper oss att förstå problemet.

    Jag. Vad är problemet och/eller säkerhetsproblem?

    II. Säkerhetsrisker, ange CVE (mitre.org) eller den fylld ut CVSS3 v3 Kalkylatorn (https://www.first.org/cvss/calculator/3.0).

    III. Finns det en lösning eller minimering? Om Ja, bör du ange steg.

    IV. Du får ett meddelande som du vill skicka till kunder? Vi kommer att fungera med dig för att skapa ett meddelande om tillämpligt.

4. Skicka bekräftelse - när du har skickat ditt problem kan vi bekräfta mottagandet inom en arbetsdag och tilldela problemet en prioritet och allvarlighetsgrad.

    - Om du behöver kommunicera med oss om problemet Använd bekräftelsenumret i alla postkommunikation.
    - Du kan visa förloppet på ditt problem när som helst.

5. Vad händer nu? Beroende på problemet och allvarlighetsgrad vidtas följande åtgärder:

    - Vi kommer att kommunicera resultatet av våra assessment till dig. Beroende på resultatet, kan vi ta bort eller begära att du ändrar dina erbjudanden. I den här händelsen fungerar det med dig för att säkerställa att avbrott för berörda kunder minimeras.
    - Vi kommer att fungera med dig för att minimera effekten av incident/säkerhetsproblem för våra ömsesidiga kunder.
