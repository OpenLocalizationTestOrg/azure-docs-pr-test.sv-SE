# <a name="data-catalog-developer-samples"></a>Data Catalog developer-exempel
Börja utveckla Data Catalog appar med hjälp av Data Catalog REST-API: et. Data Catalog REST-API: et är en REST-baserad API som ger programmatisk åtkomst till Data Catalog resurser för att registrera, kommentera och söka datatillgångar programmässigt.

Här följer exempel har vi på GitHub:

* [Kom igång med Azure Data Catalog](https://azure.microsoft.com/resources/samples/data-catalog-dotnet-get-started/)
  
  Ett exempel som visar hur du autentisera med Azure AD för att registrera, söka och ta bort en datatillgång med Data Catalog REST-API: et.
* [Massinläsning registrera och kommentera](https://azure.microsoft.com/resources/samples/data-catalog-dotnet-excel-register-data-assets/)
  
  Ett exempel som visar hur du Massredigera registrera datatillgångar från en Excel-arbetsbok med Data Catalog REST API: et och öppna XML.
* [Verktyget Import/Export](https://azure.microsoft.com/resources/samples/data-catalog-dotnet-import-export/)
  
  Ett exempel som visar hur du använder Data Catalog REST-API: et för att hämta tillgångar från Azure Data Catalog och serialisera dem till en fil. Här visas också hur du gör en uppsättning tillgångar serialiserad som JSON och skicka dem till katalogen. Det stöder exporterar en delmängd av en katalog med en sökfråga.

* [Ordlista för bulk-import](https://azure.microsoft.com/resources/samples/data-catalog-bulk-import-glossary/)

    Det här exemplet visas hur du importerar ordlista från CSV-filer till ADC ordlista.
