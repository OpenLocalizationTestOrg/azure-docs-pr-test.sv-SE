### <a name="configure-resource-based-kcd-for-a-computer-account-on-a-managed-domain"></a><span data-ttu-id="0f7a7-130">Konfigurera resursbaserade KCD för ett datorkonto på en hanterad domän</span><span class="sxs-lookup"><span data-stu-id="0f7a7-130">Configure resource-based KCD for a computer account on a managed domain</span></span>

```
Set-ADComputer contoso100-api.contoso100.com -PrincipalsAllowedToDelegateToAccount $ImpersonatingAccount
```
