# <a name="enable-the-preview-experience-to-administer-your-managed-domain-from-the-azure-portal"></a>Aktivera preview erfarenhet att administrera din hanterade domän från Azure-portalen

## <a name="your-existing-managed-domain-is-not-shown-in-the-azure-portal-preview-experience"></a>Din befintliga hanterade domän visas inte i Azure-portalen (förhandsgranskning)-upplevelse
Du har en befintlig [Azure AD Domain Services](active-directory-ds-overview.md) hanterad domän som har skapats med den klassiska Azure-portalen. När du navigerar till Azure portal innehåller bladet Azure AD Domain Services (förhandsversion) inte befintliga hanterade domänen.

![Befintliga hanterade domänen visas inte i Azure-portalen](./media/getting-started/missing-domain-services-instance.png)

Det beror på att din befintliga hanterade domän inte har flyttats över till den nya Azure-portalen. Du kan kontakta oss om du vill aktivera den nya Azure-portal (förhandsgranskning) upplevelsen för din befintliga hanterade domän.


## <a name="join-the-preview-for-the-azure-portal-experience-for-azure-ad-domain-services"></a>Ansluta till förhandsgranskning för Azure portal upplevelsen för Azure AD Domain Services
Följ instruktionerna på den [kontaktar du oss](active-directory-ds-contact-us.md) sidan och skicka e-post. Vi kan läsare att aktivera Azure portal preview för din befintliga hanterade domän.

> [!NOTE]
> När den nya Azure portal upplevelsen för Azure AD Domain Services är allmänt tillgänglig (GA), flyttas automatiskt alla befintliga hanterade domäner. Då kommer du att kunna hantera din befintliga hanterade domän från Azure-portalen.
