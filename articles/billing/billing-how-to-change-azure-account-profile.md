# <a name="change-your-azure-account-profile-information-such-as-contact-email-address-and-phone-number"></a>Ändra din profilinformation för Azure-konto som kontaktens e-post, adress och telefonnummer

E-postadressen i administratörsprofil konto används av Microsoft för att meddela dig om viktiga fakturerings-relaterade uppdateringar om prenumerationen. Vi rekommenderar att du anger en Kontakta e-postadress som du regelbundet kontrollera.

## <a name="how-to-change-your-profile-information"></a>Så här ändrar du din profilinformation

1. Logga in på den [Azure Kontocenter](https://account.azure.com/Profile).
1. Välj den **redigera detaljer** knappen och uppdatera sedan den **profil** information.

   ![Profil](./media/billing-how-to-change-azure-account-profile/profile.png)

## <a name="frequently-asked-questions"></a>Vanliga frågor och svar

### <a name="can-notifications-be-sent-to-a-different-email-address-other-than-the-account-administrator-email-address-associated-with-my-account"></a>Kan meddelanden skickas till en annan e-postadress andra kontoadministratör e-postadressen som är kopplade till mitt konto?

Ja. Den e-postadressen för kontot hämtar viktiga meddelanden om alla prenumerationer under kontot. Därför rekommenderar vi att du anger en Kontakta e-postadress som kontoadministratör söker regelbundet.

### <a name="can-i-change-the-account-administrator-email-address-in-my-profile"></a>Kan jag ändra kontoadministratör e-postadress i min profil?

Ja. Kontoadministratören är den person som ställa in Azure-konto och som tar emot viktiga e-postmeddelanden om alla prenumerationer under kontot. Vi rekommenderar att du anger en Kontakta e-postadress som kontoadministratör söker regelbundet.

### <a name="does-updating-my-profile-email-also-update-my-login-email-address"></a>Uppdatera min profil e-post också uppdateras min inloggningen e-postadress?

Nej. Uppdatera profilen e-postadress uppdateras inte din e-postadress för inloggning. Om du vill ändra din e-postadress för inloggning, måste du överför ägarskapet för kontot.

### <a name="does-updating-my-profile-address-also-update-my-credit-card-billing-address"></a>Uppdaterar min profil adressen också uppdateras mitt kontokort faktureringsadress?

Information om hur du uppdaterar din faktureringsinformation finns [ändra det kreditkort som används för att betala för en Azure-prenumeration](billing-how-to-change-credit-card.md).

### <a name="why-cant-i-update-the-country"></a>Varför kan jag uppdatera landet?

Vi kan inte ändra land för ett befintligt konto på grund av tekniska begränsningar. Du kan dock skapa ett konto i önskad land och sedan kontakta Azure-supporten om du vill migrera dina tjänster till detta konto.

### <a name="what-email-address-does-the-azure-billing-alerts-preview-feature-use"></a>Vilken e-postadress har Azure Billing aviseringar Förhandsgranska funktion används?

Du får aviseringsmeddelanden till e-postadress som du lägger till när du registrerar dig för aviseringar. E-postadressen används för aviseringar är separat från kontoadministratör e-postadress. Mer information finns i [konfigurera fakturering aviseringar för Microsoft Azure-prenumerationer](billing-set-up-alerts.md).
