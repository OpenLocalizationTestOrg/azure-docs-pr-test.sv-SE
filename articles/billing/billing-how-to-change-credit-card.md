# <a name="change-the-credit-card-used-to-pay-for-an-azure-subscription"></a>Ändra det kreditkort som används för att betala för en Azure-prenumeration
Du kan lägga till, ändra eller ta bort ett kreditkort som är kopplad till ditt Azure-konto när du loggar in på den [Azure Kontocenter](https://account.windowsazure.com/Subscriptions) som den [kontoadministratör](billing-subscription-transfer.md#whoisaa). 
 
<a id="addcard"></a>
## <a name="add-a-credit-card-as-a-payment-method"></a>Lägga till ett kreditkort som en betalningsmetod

1. Logga in på den [Azure Kontocenter](https://account.windowsazure.com/Subscriptions) som kontoadministratör.
2. Välj prenumerationen.
3. Välj **Hantera betalningssätt** till höger på sidan.
4. Välj ”+” för att lägga till ett kreditkort.
5. Välj en typ av kontokort och ange informationen som krävs.
6. Välj **nästa**. 

Om kortet är giltigt, läggs den till listan över tillgängliga betalningssätt.

Om du får ett felmeddelande när du lägger till kreditkortet [kreditkort nekades på Azure anmälan](billing-credit-card-fails-during-azure-sign-up.md).

## <a name="edit-payment-information-for-an-existing-credit-card"></a>Redigera betalningsinformation för ett befintligt kontokort
  Om ditt kreditkort hämtar förnyas och talet förblir detsamma, uppdatera den befintliga kreditkortsinformationen som upphör att gälla. Om ändringarna för ditt kreditkort eftersom kortet går förlorad, stulen eller upphört att gälla, följer du stegen i den [lägga till ett kreditkort som en betalningsmetod](#addcard) avsnitt. 

1. Logga in på den [Azure Kontocenter](https://account.windowsazure.com/Subscriptions) som kontoadministratör.
2. Välj den prenumeration som är kopplad till ditt kreditkort.</br> ![Skärmbild som visar en exempel-prenumeration som har valts.](./media/billing-how-to-change-credit-card/selectsub.png)
3. Välj **hantera betalningssätt**.</br> ![Skärmbild som visar hantera metoder betalningsalternativ markerad.](./media/billing-how-to-change-credit-card/changesub_new.png)
4. Välj betalningsmetod.
5. Välj **redigera** bredvid kreditkortsnumret.</br> ![Skärmbild som visar alternativet Redigera bredvid betalningsmetoden.](./media/billing-how-to-change-credit-card/editcard_new.png)
6. Kontrollera att typ av kontokort och kortets nummer är korrekt.
7. Välj **nästa**.

## <a name="change-the-credit-card-that-pays-your-azure-bill"></a>Ändra det kreditkort som betalar fakturan Azure

1. Logga in på den [Azure Kontocenter](https://account.windowsazure.com/Subscriptions) som kontoadministratör.
2. Välj prenumerationen.
3. Välj **Hantera betalningssätt** till höger på sidan.
4. Klicka på **Använd i stället** och välj det kreditkort som du vill använda.

## <a name="remove-a-credit-card-from-the-account"></a>Ta bort ett kreditkort från kontot
1. Logga in på den [Azure Kontocenter](https://account.windowsazure.com/Subscriptions) som kontoadministratör.
2. Välj prenumerationen.
3. Välj **Hantera betalningssätt** till höger på sidan.
4. Klicka på **ta bort** för kreditkortet som du vill ta bort.

Om ditt kreditkort associeras med andra aktiva prenumerationer från Microsoft, inte du ta bort den från din Azure-konto. Ta bort kreditkortet från alla aktiva prenumerationer som du har med Microsoft.

##  <a name="how-to-make-payments"></a>Hur du gör betalningar

* Om du har ställt in ett kreditkort eller en bankkort som din betalningsmetod görs betalningen automatiskt.
* Om du använder [en faktura](https://azure.microsoft.com/pricing/invoicing/), skickar du betalningen till platsen som anges längst ned på fakturan.
