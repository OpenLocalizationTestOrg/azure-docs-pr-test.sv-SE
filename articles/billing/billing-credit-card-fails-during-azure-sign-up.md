# <a name="your-debit-card-or-credit-card-is-declined-at-azure-sign-up"></a>Din bankkort eller kreditkort har nekats på Azure-registrering
Om kreditkort eller bankkort är avvisade eller accepterades inte när du registrerar dig för Azure, kan du hantera ett av följande problem:

* **Provider för kreditkort eller debet kort accepteras inte för ditt land.** När du väljer ett kort kan se du bara de alternativ som är giltig i det land som du väljer. Kontakta din bank eller kort utfärdare för att kontrollera att ditt kreditkort har aktiverats för internationella transaktioner. Se stöds länder och valutor i [Azure köp FAQ](https://azure.microsoft.com/pricing/faq/).
* **Din kredit- eller kortinformation är felaktiga eller ofullständiga.** Namn, adress och CVV-koden som du anger måste matcha exakt vad skrivs ut på kortet.
* **Du använder en virtuell eller förbetalt kort.** Virtuella eller förbetalt kort för kredit- eller inte är godkänd som betalning för Azure-prenumerationer.
* **Kortet är inaktiva eller blockerade.** Kontakta din bank för att säkerställa kortet är aktiv.
* **Du kan ha problem med andra registreringsproblem** kolla hur man [felsöka Azure anmälan](billing-troubleshoot-azure-sign-up-issues.md).

När du har uppdaterat din betalningsinformation försök att logga igen.

## <a name="representing-a-business-that-doesnt-want-to-pay-by-card-set-up-invoicing"></a>Representerar ett företag som inte vill betala av kortet? Ställ in fakturering
Om du representerar ett företag betalar du för din Azure-prenumeration med faktura betalningssätt som kontroller, ytterligare kontroller eller elektronisk överföring. Du kan inte ändra till ett annat betalningsalternativ efter att du har konfigurerat att betala per faktura. För att betala av fakturan finns [Azure Billing - hur faktureras](https://azure.microsoft.com/pricing/invoicing/).

## <a name="update-your-credit-card-or-debit-card-information"></a>Uppdatera ditt kreditkort eller debet kortinformation
När du registrerar dig, [hantera din betalningsinformation](billing-how-to-change-credit-card.md) ändra eller ta bort ett kort.
