# <a name="paas-services-for-azure-stack"></a>PaaS-tjänster i Azure-stacken

Du kan distribuera [plattform som en tjänst](https://azure.microsoft.com/overview/what-is-paas/) (PaaS)-tjänster i Azure-stacken från Microsoft och andra 3 part-leverantörer. Om du vill få meddelanden om nya tjänster, följer du #AzureStack på Twitter.

[Lägga till en App Service-resursprovider i Azure Stack](https://docs.microsoft.com/azure/azure-stack/azure-stack-app-service-overview)

[Lägga till en provider för SQL Server-resurs i Azure stapel](https://docs.microsoft.com/azure/azure-stack/azure-stack-sql-resource-provider-deploy)

[Lägga till en resursleverantör MySQL-Server i Azure stapel](https://docs.microsoft.com/azure/azure-stack/azure-stack-mysql-resource-provider-deploy)
