# <a name="sql-database-documentation"></a>Dokumentation om SQL Database

Azure SQL Database är en relationsdatabas-som-tjänst som bygger på Microsoft SQL Server-motorn. SQL Database är en högpresterande, tillförlitlig och säker databas som du kan använda för att skapa datadrivna program och webbplatser i valfritt programmeringsspråk, utan att behöva hantera infrastrukturen. Lär dig hur du använder SQL Database med våra snabbstarter, självstudier och exempel.

## <a name="5-minute-quickstarts"></a>Snabbstarter på 5 minuter

Lär dig att skapa en SQL-databas, hantera den med hjälp av SQL Management Studio och utveckla den i C#, Java, Python, Node.js och Ruby:

- [Azure Portal](/azure/sql-database/sql-database-get-started-portal)
- [Azure PowerShell](/azure/sql-database/sql-database-get-started-powershell)
