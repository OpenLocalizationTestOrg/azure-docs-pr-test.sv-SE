# <a name="azure-database-for-mysql-documentation"></a>Azure Database för MySQL-dokumentation

Azure MySQL-databas är en relationsdatabas baserat på hello MySQL Server engine med öppen källkod.  Det är en fullständigt hanterad databas som tjänst, för hantering av verksamhetskritisk arbetsbelastning med förutsägbara prestanda och dynamisk skalbarhet. Utveckla program med Azure-databas för MySQL användning hello öppen källkod verktyg och plattform du önskar. Lär dig hur toouse Azure-databas för MySQL med vår Snabbstart, självstudier och exempel.
