| SKU | Beskrivning |
|---|---|
| Basic | En optimerad för kostnad startpunkten för utvecklare som Lär dig hur Azure-behållare registret. Grundläggande register har samma programmässiga funktioner som Standard och Premium (integrering med Azure Active Directory-autentisering, bild borttagning och web hook), men det finns begränsningar för storlek och användning. |
| Standard | Standard registret har samma funktioner som grundläggande, men med ökad Lagringsgränser och avbildningen genomflöde. Standard register ska uppfylla behoven hos de flesta scenarier för produktion. |
| Premium | Premium-register har högre gränser på begränsningarna, t.ex. lagrings- och samtidiga åtgärder, inklusive förbättrade funktioner för att stödja stora mängder. Förutom högre avbildningen genomflödeskapaciteten Premium lägger till funktioner som geo-replikering för att hantera ett enda register över flera regioner underhålla nätverket Stäng register i varje distribution. |
